# 빠른 시작 가이드 🚀

## 5분 안에 시작하기

### 1. 개발 서버 실행

```bash
cd todo-manager
npm install  # 처음 한 번만
npm run dev
```

브라우저에서 `http://localhost:5173` 열기 (또는 표시된 포트)

---

## 기본 사용법

### 첫 할일 만들기 📝

1. **"할일 추가"** 버튼 클릭
2. 제목 입력 (필수)
3. 선택사항 입력:
   - 메모: 상세 설명
   - 날짜: 기한 날짜
   - 시간: 기한 시간
   - 폴더: 분류 (업무/개인/공부 중 선택)
4. **"추가"** 버튼 클릭

### 폴더 만들기 📁

1. 좌측 사이드바에서 **"폴더 관리"** 클릭
2. 폴더 이름 입력
3. 마음에 드는 파스텔 색상 선택
4. **"폴더 추가"** 클릭

### 카테고리로 보기 🔍

상단 탭 클릭으로 할일 필터링:

- **📋 전체**: 모든 할일
- **📅 오늘**: 오늘 마감
- **📆 이번주**: 이번 주 마감
- **🔮 나중에**: 그 이후

### 할일 관리하기 ✅

- **완료**: 체크박스 클릭
- **수정**: ✏️ 아이콘 클릭
- **삭제**: 🗑️ 아이콘 클릭

---

## 모바일에서 사용하기 📱

1. 좌측 상단 **☰** 버튼으로 메뉴 열기
2. 폴더 선택
3. 화면 밖 터치로 메뉴 닫기

---

## 기본 폴더

처음 시작하면 3개의 기본 폴더가 생성됩니다:

- 🌸 **업무** (핑크)
- 💜 **개인** (라벤더)
- 💙 **공부** (민트)

원하는 대로 수정하거나 새 폴더를 추가하세요!

---

## 데이터 저장

모든 할일과 폴더는 **브라우저의 로컬 스토리지**에 자동 저장됩니다.
- ✅ 새로고침해도 데이터 유지
- ✅ 브라우저를 닫아도 데이터 유지
- ⚠️ 브라우저 데이터 삭제 시 할일도 삭제됨

---

## 팁 💡

### 효율적인 할일 관리

1. **아침에 "오늘" 탭 확인**: 오늘 할 일 체크
2. **주초에 "이번주" 탭 확인**: 주간 계획 세우기
3. **폴더별 색상 활용**: 한눈에 구분 가능
4. **메모 기능 활용**: 세부 사항 기록

### 폴더 활용 예시

- **🎯 업무**: 회의, 보고서, 이메일 답장
- **🏠 개인**: 장보기, 약속, 청소
- **📚 공부**: 강의 듣기, 과제, 독서
- **💪 운동**: 헬스, 요가, 런닝
- **🎨 취미**: 그림, 음악, 사진

## 문제 해결 🔧

### 할일이 안 보여요
- 올바른 카테고리/폴더를 선택했는지 확인
- "전체" 탭에서 모든 할일 확인

### 데이터가 사라졌어요
- 브라우저 데이터를 삭제했는지 확인
- 같은 브라우저를 사용하고 있는지 확인
- 시크릿/프라이빗 모드는 데이터가 저장되지 않음

### 페이지가 느려요
- 브라우저 캐시 삭제
- 페이지 새로고침 (F5)
- 오래된 완료 할일 정리
