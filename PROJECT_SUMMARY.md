## 🏗️ 기술 아키텍처

### 컴포넌트 구조

```
App.jsx (메인)
├── FolderSidebar.jsx (폴더 관리)
│   └── FolderManager.jsx (폴더 추가/수정/삭제)
├── Dashboard.jsx (오늘의 일정)
├── CategoryTabs.jsx (필터 탭)
├── TodoList.jsx (할일 목록)
│   └── TodoItem.jsx (개별 할일)
└── TodoForm.jsx (할일 추가/수정 모달)
```

### 커스텀 훅

1. **useLocalStorage**: 로컬 스토리지 관리
2. **useTodos**: 할일 CRUD 로직
3. **useFolders**: 폴더 CRUD 로직
4. **useDateFilter**: 날짜 기반 필터링

### 스타일 시스템

1. **variables.css**: CSS 변수 정의
2. **global.css**: 전역 스타일 & 유틸리티 클래스
3. **animations.css**: 재사용 가능한 애니메이션
4. **Component.css**: 각 컴포넌트별 스타일

## 🎯 핵심 기능 동작 방식

### 1. 할일 추가 플로우
```
사용자 클릭 "할일 추가" 
→ TodoForm 모달 열림 
→ 데이터 입력 
→ useTodos.addTodo() 호출 
→ 로컬 스토리지 자동 저장 
→ UI 자동 업데이트
```

### 2. 카테고리 필터링 플로우
```
사용자 탭 선택 
→ useDateFilter 훅 실행 
→ date-fns로 날짜 비교 
→ 필터링된 할일만 표시 
→ 정렬 (날짜순, 완료 상태별)
```

### 3. 폴더 시스템 플로우
```
폴더 선택 
→ selectedFolderId 상태 변경 
→ useDateFilter가 폴더별 필터링 
→ 해당 폴더의 할일만 표시 
→ 폴더 색상 자동 적용
```
